## Notes

I want to ship a new design. I added HTML mockups generated by Claude Design. Adopting the new HTML requires two functionality changes:

1. **Task tags** — each task can have categories added as tags. You type something, press Enter, and a tag is added. Tags already used in the same household are suggested (autocomplete).
2. **Emailed invite link** — added an option to send the household invite link by email. The templates include emails for both password reset and the invite link.
